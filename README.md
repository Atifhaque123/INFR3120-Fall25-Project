# Event Scheduler – Project Part 1

This web application lets users create, view, edit, and delete events.
It is built using basic HTML, CSS, and JavaScript with localStorage for data storage.

## Features (Part 1)
- Landing page with site name and team logo
- Hero button to create a new event
- Table listing all created events
- Full CRUD operations:
  - Create event
  - Read/List events
  - Update event
  - Delete event

## Technologies Used
- HTML
- CSS
- JavaScript (localStorage)

## Group Members

## How to Use
1. Open index.html in a browser.
2. Click "Create Event" to add a new event.
3. Use Edit/Delete buttons to modify or remove an event.
4. Data stays saved using localStorage.
